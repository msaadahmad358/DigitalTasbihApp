# Digital Tasbih

A beautiful and fast Islamic Dhikr counter app for counting daily tasbih.

## Features

- 🕌 **3 Default Tasbihs** - Subhan Allah, Alhamdulillah, Allahu Akbar
- ➕ **Add Custom Tasbihs** - Create your own with custom target counts
- ✨ **Tap to Count** - Increment with haptic feedback
- 🔄 **Decrement & Reset** - Fix mistakes easily
- 📊 **Progress Indicator** - Visual tracking of your goal
- 🎯 **Set Custom Targets** - Adjust target count for any tasbih
- 💾 **Auto-Save** - Never lose your counts
- 🎨 **Beautiful UI** - Gradient design with smooth animations
- 📱 **Responsive** - Works on all screen sizes

## Tech Stack

- Flutter 3.x
- Dart 3.x
- Shared Preferences for local storage
- Device Preview for testing
